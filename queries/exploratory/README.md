# Exploratory Queries
During the initial exploration of the datasets, SQL queries were used to:

Count the number of countries in each dataset.

Identify duplicate records.

Correct misspellings in Data

Verify available years and select 2024 observations.

Confirm salary units (USD PPP).

Validate data before joining the datasets.
 
The corresponding SQL scripts are available in sql/02_exploratory_queries.sql
