# Final Queries

Production-ready SQL queries developed to answer the project's business questions and support the Power BI dashboard.

The final queries include:

Ranking countries by Salary Value Score using a Common Table Expression (CTE) and the RANK() window function.

Identifying the top 10 countries by Average Annual Salary.

Identifying the top 10 countries by the Cost of Living Index.

Ranking countries by Local Purchasing Power Index.

Generating the KPI metrics used within the dashboard.

Categorising countries based on Salary Value Score using conditional logic (CASE) or quartiles (NTILE()).
